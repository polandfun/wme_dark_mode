Enable a better dark mode for WME.

Features -
- Custom CSS to tailor various UI elements to look good in dark mode.
- Currently supporting majority of popular plugins.
- Modifies WME/Beta/Profile pages.
- Includes a dark/light mode toggle under the profile and WME Settings.
- Dark/Light mode option is synced across all open tabs.
- Light mode is unmodified from stock.

If a plugin is found to be broken or look bad, feel free to ask on dark mode discord, or on discuss - <link_pending>

Warning - If FUME is used, the UI Contrast Enhancement needs to be set to 'None'. A warning message is included in the script to detect this circumstance.
